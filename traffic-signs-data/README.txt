This folder contains a provided link to download data to train, validate and test the model. The data are on pickle format (.p) and contains : 

- 34799 images for the training set
- 4410 images for the validation set
- 12630 images for the test set

The images' shapes are (32, 32, 3).

Link : https://github.com/fredotran/traffic-sign-recognition/releases/download/1.0/traffic-signs-data.zip
